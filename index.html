jQuery Calendar Canvas Plugin

Calendar Icons.

Download this project as a .zip file Download this project as a tar.gz file

About

This plugin can be used to create calendar icons similar to the iOS Calendar app. An example with various sizes and the different initialization codes can be found in example.htm.

Basic Usage

// in your head...
$(document).ready(function() {
  $('li').calendarize({size: 16});
});

// somewhere else...
<li data-calendar="June 15">Buy milk.</li>

Configuration options and defaults

var defaults = {
  topBackColor   : "#f01",
  topFrontColor  : "#fff",

  mainBackColor   : "#fefefe",
  mainFrontColor  : "#000",  
  borderColor     : "#000",  

  topFrontShadowColor  : "#333",
  mainBackShadowColor  : "#ccc",
  mainFrontShadowColor : "#000",

  preserveContent : true,

  size      : 32,
  shadow    : 0.1,

  topFont  : "Verdana",
  mainFont : "Arial"
};
